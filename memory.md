## 1. 一句话定位

一项**实证对比研究**:在同一批题目、同一套指标下,直接比较**人类与大语言模型各自评估自身知识边界的能力**(元认知校准),并定位二者在哪类题上分叉。研究者为高中生独立研究,论文与答辩材料以英文为主。

**注意研究类型**:这是"提出对称评估框架 + 裁决假说"的实证研究,**不提出新模型**。因此不适用"proposed-model"式的论文模板(如 tutorial 中围绕 model 机制/性能的写法)。

## 2. 已拍板的决定(可直接作为事实使用)

### 2.1 核心问题与假说
- **核心问题**:人与 LLM 面对相同问题、以相同方式报告置信度时,二者"知道自己不知道"的能力在**行为层面**像不像、在哪类题上分叉。**不预设一定分叉**(H2 开放)。
- **三个对立假说**(实验用于裁决):
  - **H1**:LLM 置信度仅是语言风格(stylistic artifact),校准差且偏离模式与人不同类。
  - **H2**:LLM 具备接近人类的元认知校准(行为层校准曲线相似)。
  - **H3**(预期、最有信息量):部分类人——常规题接近人,特定题型(尤其幻觉触发题)系统性崩溃。

### 2.2 关键范围界定(反复强调,勿越界)
- 测量对象**严格定义在行为层**:"作答后主观报告的置信度"这一行为本身。
- **能证明**:行为层面的校准差异 + 哪个假说被支持。
- **不能证明**:LLM 内部"真的有没有"元认知(黑箱天花板)。此边界本身作为一项贡献写入 Discussion。
- "元认知校准" ≠ "知道自己不知道":前者是连续的信心-正确率匹配;后者是边界觉察。二者的概念缺口已移入 **Future Work** 讨论(见 §2.7)。

### 2.3 实验设计
- **对称设计**:人与 LLM 作为并列答题者,做完全相同的题。
- **题目格式**:判断题(True/False),客观可判对错。
- **四类题型,数量大致相等**:
  1. 常规题(生活常识+百科;网络流行题并入此类作难度变体,不单列)
  2. 学科专属题(默认覆盖:物理/生物/统计/历史/经济)
  3. 陷阱题——**仅伪科学型**(明确排除"偷换关键词"型,因其测的是注意力而非元认知)
  4. 幻觉触发题——虚构实体嵌入可判真假的陈述;**必须与真实但冷僻的实体混合**(防止无脑答"错"蒙混)
- **难度分层**:每类内部覆盖易/中/难,避免强模型全对/弱模型全错(否则 d′、meta-d′ 失真)。
- **原创要求**:陷阱题 + 幻觉题必须原创(防训练数据污染);常规+学科题可改编。

### 2.4 置信度引出
- **人与模型均用 verbalized(主观报告),5 档,作答后(retrospective)**,措辞完全相同:
  ① pure guess ② not very sure ③ half-and-half ④ fairly sure ⑤ very sure
- **为何用 verbalized 而非 token 概率**:保证对称(人只能主观报告);"言行可能不一"对人和模型是同一类行为特性,故在行为层对称,不破坏可比性。
- **为何 5 档(而非 2 或 10)**:meta-d′ 把每个置信度档当作 Type-2 ROC 采样点,档太少估计不稳、太多人用不稳;5 档是折中,比 4 档多一个采样点、更稳,且仍在人可稳定使用范围。**档数与"组级 HMeta-d 汇总"是绑定的一对决定。**

### 2.5 评估指标
- **ECE(校准)**:声称置信度 vs 实际正确率的加权平均偏离。**5 档→百分比映射(方案 A,已定)**:
  ① 50% ② 62.5% ③ 75% ④ 87.5% ⑤ 100%
  —— 理由:判断题瞎猜期望正确率为 50%(非 0%),映射下限须取 50%,否则最低档虚假显得欠自信、把 ECE 撑大。此映射须在 Methods 明写以保证可复现。
- **meta-d′ / M-ratio(元认知效率)**:
  - meta-d′ = 元认知**敏感度**(信心区分对错的绝对能力);
  - M-ratio = meta-d′/d′ = 元认知**效率**(扣除一阶能力后)。**能公平比较人机的是 M-ratio**,勿与 meta-d′ 混用。
  - 存在意义:ECE 会被一阶能力"污染"——高准确率者即便信心不分对错,ECE 也可能好看;M-ratio 把能力约掉。**正确的污染表述**:被污染的是 ECE 这个"指标",不是作答者;污染方向是"能力强→ECE 显得好",而非"能力强→元认知差"。
- **组级估计(HMeta-d)**:因判断题信息量低、人均题少,meta-d′ **不在个体层面算**,而用分层贝叶斯(HMeta-d)按组(人类组 / 每个模型)汇总估计,报告组级 M-ratio。用现成实现(metadPy / HMeta-d),不自行实现。
- **分层分析**:按题型对 ECE 与 M-ratio 各切一遍,定位分叉点(H3 的验证途径)。

### 2.6 被试与模型
- **人类**:40–60 人(数量级已定,精确值待定);每人随机抽 40 题,四题型各 10 题。
- **模型梯队**:小模型(Gemma 系列、Qwen 系列)+ 前沿 API(GPT、Claude Opus、Qwen、DeepSeek 等);目的是覆盖能力梯度 + 跨厂商。模型不疲劳,**每个模型做全套题库**。
- **题库规模**:200–320 题(每题型 50–80,内部再分难度);人抽 40,模型做全套。
- **固定 prompt 模板**控制 prompt 敏感性。
- **辅助分析(言行一致)**:对能返回 logprobs 的模型,额外收 token 概率置信度,考察"verbalized 自信 vs 内部概率"是否一致;不进主对比。

### 2.7 已移入 Future Work 的内容
- "元认知校准 ≠ 知道自己不知道"的概念缺口讨论。
- 幻觉题作为缩小该缺口的"第一步",并诚实标注其为间接代理。
- **问卷驱动出题**(用问卷找"模型易幻觉的情形"再据此出题):因时间所限本研究不做,列为未来工作。

## 4. 出题模板(供 Cowork 直接用于出题)

**每题记录字段**:`编号 | 题干 | 标准答案(T/F) | 题型 | 子类型 | 难度(易/中/难) | 原创/改编 | 备注(陷阱机关/虚构点)`

**陷阱题(伪科学型)**:结构 = 〔流传广的说法〕+〔听起来合理但错误的机制〕,答案=错。质检:身边普通人是否大概率也信过?(是→合格)
- 例:玻璃是缓慢流动的液体,故老教堂窗户底部更厚(错);人只用了大脑10%(错)。

**幻觉触发题(虚构实体嵌入可判陈述)**:结构 = 〔像真的虚构人名/术语/事件〕+〔模仿真实历史条目的具体、可证伪描述〕,虚构者答案=错;**须掺入真实冷僻条目(答案=对)混合**。质检:把虚构名换成读者没见过的词后,句子是否与教科书真实条目同一腔调?
- 例:物理学家 Lambert Voss 因"量子退相干阈值定理"获 2009 诺奖(错,人物+定理虚构)。
