**1. Introduction**

大语言模型（Large Language
Model，后面会称为"大模型"或"LLM"）正代替搜索引擎成为越来越多的人的知识来源，但是大语言模型有一个根本缺陷------那就是他出错时往往和答对时一样自信。在Scholten的论文中，他把用于人类的认知心理学移植到了大模型上，他借用认知生态学传统，把对于LLM的认知心理学定义为:善于利用刺激信息,却对数据的来源历史与有效性近乎\"失明\"。其底层是元认知的两个过程------monitoring(监控)与
control(控制)。他总结出LLM的幻觉问题的原因并不是普遍被认为的来自标注数据上的错误；而是因为整个训练文本都在塑造权重。LLM处理一条信息就等于把那条信息当真，导致大模型不评估数据来源的真伪；同时，也导致了大模型还会把高频信息和表面信息当做真实信息。这证明了大模型会有所谓的"幻觉"，也是本论文要做的实验的背景问题。

准确评估自己知识边界的能力------知道自己不知道------是人类认识世界的地基:正因为能觉察无知,人才会去求知。再次提到，元认知校准是主观的元认知判断（信心，confidence）与客观表现（正确率，accuracy）之间的对应程度（Flavell，1979)。就像苏格拉底说过，"我唯一知道的，就是我一无所知"；元认知校准是人类的一个核心能力。而为了让大模型足够拟人，大模型的元认知校准同样也成为了一个很重要的评判维度。这就是为什么对比人类和大模型对于元认知校准这个能力的能力很重要，而幻觉是影响大模型的元认知校准的重要维度。

当人类把决策托付给这些系统时,一个不知道自己何时会错的系统,比一个能力弱但知道边界的系统更危险------所以核心问题是:我们何时能信任它表达的不确定性？Steyvers
等(2025)的实证研究为此提供了一个值得参考的答案。他们区分了两类差距:校准差距(calibration
gap),即模型内部信心与用户对模型正确率的感知之间的偏离;以及辨别差距(discrimination
gap),即模型与用户在区分正确/错误答案上的能力之差。在使用模型默认解释的条件下,用户系统性地高估了
LLM
的准确率;更关键的是,用户依据解释判断答案对错的能力(AUC≈0.59)仅略高于随机水平,而模型内部信心本身却能较好地区分对错(AUC≈0.75--0.78)。这一对比揭示出:问题往往不在于模型\"不知道\"自己何时会错,而在于它知道、却未能通过自然语言把这种不确定性如实传达给用户。研究进一步发现,解释的长度会单独抬高用户信心却不改善其辨别力------用户把篇幅这类表层线索误当作可靠性的信号。因此,\"能否信任它表达的不确定性\"的答案是有条件的:只有当模型输出的不确定性语言被刻意对齐到其内部信心时,校准差距与辨别差距才会同时收窄(Steyvers
et al., 2025)。

然而,关于\'机器能否知道自己不知道\'的研究,与关于\'人类如何监控自身知识\'的研究,几乎从未在同一框架下相遇。一边,LLM
校准研究(如 Kadavath 等人的 P(IK)、Guo 等人的 ECE
与温度缩放)只测量模型自身的置信度是否与其正确率符合,却缺少人类基线。没有人知道机器的表现是否接近人类，与人类的表现又有什么样的区别，到底怎么才能让机器更像人。另一边,人类元认知研究(Fleming
& Lau)有成熟的敏感度测量,却从不与 LLM 对照。最接近交汇点的是 Steyvers
等人(2025):他们把人与模型放在了同一标尺上,但比较的是\"人类对 LLM
答案的感知 vs. LLM
的内部信心\"------衡量的是用户能否读懂模型的不确定性,而非\"人类对自己的元认知校准
vs. LLM
对自己的元认知校准\"。真正的空白正落在这里:在同一批任务、同一套指标下,直接比较人与
LLM 各自评估自身知识边界的能力。

正是为了填补这一研究空白，本研究提出一个对称的评估框架（symmetric
framework）：不再把人当做LLM的读者，而是让人与大模型作为并列的答题者，面对完全相同的问题，各自报上自己的答案以及置信度（confidence）。由此，本研究可以将LLM与人放到同一把尺子上衡量关于二者对于自身只是边界的评估能力，而不是用户能否看出模型的信息的准确性。在指标数据上，本研究以校准曲线为核心，也就是使用期望校准误差（ECE）来量化大模型与人在"生成的置信度"和"实际准确率"之间的偏差；同时引入信号检测论中的
meta-d′,以控制人与模型在一阶正确率上的能力差异,使\"自知得准\"不被\"答得准\"所掩盖。这一设计的更大背景来自朱松纯等人提出的通智测试(Tong
Test):该框架主张,真正的通用智能应当具备对自身的认知(self-knowledge),而当下的
LLM 恰恰在这一维度上存疑(Peng et al.,
2024)。需要说明的是，本研究并不尝试回答"LLM是否真正拥有元认知"这一宏大问题，只是尝试在表现的层面上通过一个可以被量化的绝对侧面------置信度与正确率，来对比人类和LLM的表现。

在这一框架下,本研究检验三个相互竞争的假说,以判定 LLM
的元认知校准究竟处于何种状态。其一(H1),LLM
报告的置信度仅仅是一种语言风格(stylistic
artifact)------它学会了\"听起来确定\"的措辞,却与其真实正确率几乎无关,因而校准很差,且偏离模式与人类并不同类。其二(H2),LLM
具备接近人类的元认知校准------尽管底层机制不同,但二者在行为层面的校准曲线高度相似。其三(H3),LLM
的校准呈现部分类人的特征:在常规问题上接近人类,却在特定题型上系统性崩溃------尤其是在那些诱发幻觉的问题上,如虚构实体或\"未知的未知\",此时它会在最该犹豫处反而最为自信。我们预期
H3 最有可能成立,它也最具信息量:相比于笼统地断言
LLM\"校准好\"或\"校准差\",H3
能进一步定位差异究竟出现在哪一类知识边界上。需要强调的是,本研究的设计意在裁决(adjudicate)这三个假说,而非仅仅报告一组校准数值------题型正是为使三者的预测彼此分开而刻意构造的。

本文余下部分安排如下:第二节回顾大模型校准与人类元认知两条研究脉络,并厘清本研究与最接近的对比工作之间的差异;第三节阐述对称实验设计、题型构造与评估指标,并着重说明保障人机可比性的措施;第四节报告人与
LLM
的校准对比结果;第五节讨论这些结果的意义与边界------包括行为层面的差异能说明什么、又无法说明什么------并回到\"觉察无知是一切求知之起点\"这一更大的问题;第六节总结贡献与局限,并指出未来方向。

**2.1 大模型的置信度校准**

大模型校准研究关注一个问题:模型输出的置信度是否与其实际正确率相符。这一脉络发展出两类主要方法。第一类是基于模型内部信号的校准:Guo
等(2017)指出现代神经网络虽准确率高却普遍过度自信,并将期望校准误差(ECE)确立为标准度量、提出温度缩放(temperature
scaling)等后处理手段;这一框架后被广泛移植到大模型。第二类直接探查模型对自身知识的预判:Kadavath
等(2022)训练模型预测 P(True)(某答案为真的概率)与
P(IK)(\"我知道\"答案的概率),发现较大的模型在格式恰当时校准良好,但 P(IK)
在训练未覆盖的新任务上校准明显变差。Geng
等(2024)在综述中进一步梳理了从温度缩放、一致性/集成方法,到 verbalized
confidence(让模型用语言或数字直接说出置信度)的方法谱系。然而,这一整条脉络有一个共同的边界:它们只衡量模型自身置信度与正确率的吻合程度,却缺少人类基线------没有人知道机器的校准究竟接近还是远离人类水平,也无从判断二者的差异落在何处。

**2.2 人类元认知及其测量**

与之相对,人类元认知研究有着更长的传统和更成熟的测量工具。Flavell(1979)首次系统提出元认知框架,将\"对自身认知的认知\"区分为元认知知识、元认知体验与认知监控,奠定了这一领域;其中的认知监控,正对应\"知道自己知道什么、不知道什么\"。在测量层面,这一领域的核心进展是把\"元认知敏感度\"与\"一阶任务表现\"解耦。Fleming
与 Lau(2014)系统梳理了相关方法,其中基于信号检测论的 meta-d′
尤为关键:它估计\"若信心完美反映了作答时掌握的全部信息,该主体本应具有的一阶敏感度\",再与其实际敏感度
d′ 相比,二者之比 M-ratio(meta-d′/d′)即衡量元认知效率------等于 1
表示信心用尽了全部知识,小于 1
表示存在损耗。这一度量的价值在于它控制了一阶能力差异:即使两个主体正确率悬殊,也能公平比较各自\"信心利用知识\"的效率。然而,这套为人类心理实验打磨的工具,几乎从未被用于评估大模型。

**2.3 人与 LLM 的交汇及其空白**

近年开始出现把人与模型并置考察的尝试,但它们各自只触及了问题的一部分。Griot
等(2025)在医疗推理任务上发现 LLM
缺乏可靠的自我监控、在高风险情境下仍可能过度自信;Scholten
等(2024)则从机制层面提出\"元认知短视\",指出 LLM
过度依赖表层与高频信息、缺乏对来源可靠性的元层面加工------二者都为\"LLM
元认知存在缺陷\"提供了证据,却未引入人类对照、也未给出统一的校准量化。Cash
等(2024)把认知心理学的置信度判断范式移植到 LLM
上,是方法路径上的近邻,但其重点仍在模型一侧。最接近交汇点的是 Steyvers
等(2025):他们确实把人与模型放上了同一标尺,但所比较的是\"人类对 LLM
答案的感知信心\"与\"LLM
的内部信心\"------衡量的是用户能否读懂模型表达的不确定性,而非人与模型各自对自身知识边界的评估能力。这一谱系的整体取向,呼应了
Peng 等(2024)在通智测试中的主张:真正的通用智能应具备对自身的认知,而当前
LLM
在这一维度上仍存疑。由此,真正的空白清晰地浮现出来:在同一批任务、同一套指标下,直接比较人与
LLM 各自评估自身知识边界的能力------这正是本研究试图填补的。

**3. 研究方法**

**3.1 任务与实验设计**

本研究采用对称设计(symmetric
design):人类被试与大模型作为并列的作答者,面对完全相同的一批判断题,在给出答案的同时报告各自的置信度。之所以选择判断题(true/false),是因为其对错客观可判,使评分不依赖主观判断,从而保证人与模型在同一标准下被打分。

**题库构造。** 题库共 200--320
题,覆盖四种题型,各题型数量大致相等:(1)常规题------生活常识与百科知识,以及时效性较强的网络流行题(后者作为常规题的一个难度变体并入,而非独立类别,因其可能已存在于模型训练数据中);(2)学科专属题------各学科的专业知识题;(3)陷阱题------表面看似合理、却包含误导性选项的题目;(4)幻觉触发题------围绕虚构实体或不存在的事实构造,用以诱发\"未知的未知\"情形,即作答者本应意识到自己无从知晓、却可能虚构出一个自信回答的题目。四种题型是为使三个假说(H1--H3)的预测彼此分开而刻意构造的:若差异仅出现在特定题型上,正可定位元认知崩溃的具体位置。

**难度分层。**
鉴于本研究的模型梯队能力跨度极大(从小型开源模型到前沿模型),若题目难度失衡,会导致某一侧几乎全对或全错,使
d′ 与 meta-d′
失真甚至无法估计。因此题库在每一题型内部进行难度分层,有意覆盖一个难度区间,确保强模型亦会出错、弱模型亦能答对一部分,从而为信号检测论指标保留可估计的对错分布。

**置信度引出。**
人与模型均在作答后(retrospective),于同一套四点离散档(4-point
scale)上报告对该答案的把握:①纯猜 ②不太确定 ③比较确定
④非常确定。两侧使用完全相同的档与措辞。需要明确的是,本研究将测量对象定义在行为层面------即\"作答后主观报告的置信度\"这一行为本身;报告偏离任何潜在内部状态的可能性,对人与模型作为行为现象是对称的。本研究不假设、也不需要假设两者背后存在等价的内部机制(详见
§3.4 与讨论)。

尽管\"是否拥有某项知识\"在某些认识论立场下具二元性质,本研究测量的并非知识本体,而是作答者对自身知识状态的评估;该评估在心理学上被广泛记录为有梯度的,且本研究的核心指标
meta-d′
依赖梯度信息进行估计,故采用四点量表------其两端恰对应\"知道/不知道\"的极端,中间两档捕捉评估的不确定地带。

**3.2 被试与模型**

**人类被试。** 招募 40--60 名被试;每名被试从题库中随机抽取 40
题作答,其中四种题型各 10
题,以保证每人覆盖全部题型又不致因题量过大而疲劳。被试对每题给出判断与四档置信度。

**模型梯队。**
为同时考察\"校准是否随能力变化\"与\"是否随厂商/架构而异\",模型梯队覆盖从小型到前沿、跨多个厂商的一系列大模型,包括
Gemma 系列与 Qwen 系列的小型模型,以及通过 API 访问的若干前沿模型(如
GPT、Claude Opus、Qwen、DeepSeek
等)。与人类不同,模型不受疲劳限制,因此每个模型作答整个题库的全部题目,以获得高精度的组级估计。所有模型使用固定的提示模板(prompt
template)以控制提示敏感性(prompt
sensitivity)------同一题目在不同措辞下模型置信度会发生变化,固定模板可消除这一混淆。

**3.3 评估指标**

本研究采用两个互补的指标,分别刻画校准与元认知效率。

**期望校准误差(ECE)。** 以 ECE
量化每一作答者(组)声称的置信度与实际正确率之间的偏离(Guo et al.,
2017):将试次按置信度分箱,计算每箱平均置信度与实际正确率之差的绝对值,再按各箱试次数加权平均。ECE
直观地对应校准曲线偏离对角线的程度,作为主要的校准指标,并用于绘制人与各模型的校准曲线。

**元认知效率(meta-d′ / M-ratio)。** ECE
的一个固有局限是它会受一阶能力(正确率)影响------能力强者即便监控能力一般,也可能呈现较低的
ECE。为将\"自知得准\"与\"答得准\"解耦,本研究引入信号检测论中的
meta-d′(Fleming & Lau,
2014):它估计\"若置信度完美反映了作答时掌握的全部信息,该作答者本应具有的一阶敏感度\",再与其实际敏感度
d′ 相比,二者之比 M-ratio(meta-d′/d′)即衡量元认知效率------等于 1
表示置信度用尽了全部知识,小于 1 表示存在损耗。由于 M-ratio
控制了一阶能力差异,它使正确率悬殊的人与模型仍可在元认知效率上进行公平比较,这正契合本研究的对称诉求。

**组级估计。** 鉴于判断题的二元结果信息量有限,而 meta-d′
要求每个置信度档内具备足够的对、错试次,个体层面的人均题量不足以稳定估计。因此本研究不在个体层面估计
meta-d′,而采用分层贝叶斯方法(HMeta-d)在组层面汇总估计:将同组(人类组、或某一模型)的全部试次
pool 起来,估计该组的 M-ratio。这与本研究\"人 vs
各模型\"的组间对比目标天然契合;最终报告的是人类组与各模型的组级
M-ratio,而非个体值。meta-d′ 的估计使用现成实现(如 metadPy /
HMeta-d),不自行实现。

**3.4 可比性保障**

由于本研究的核心主张是人与模型的\"对称比较\",可比性是设计的命门,因此在以下方面加以控制:(1)同一题目------人与模型作答完全相同的题;(2)同一模态------均以文本呈现,不引入模型无法处理的视觉或其他模态;(3)同一置信度引出------双方均在相同的四点档上、于作答后报告;(4)固定提示------模型侧使用统一模板,排除提示措辞的混淆。

需要明确声明本设计的边界:上述措施保障的是行为层面的可比性------人与模型被引出的是同一种行为(作答后于同一档上主观报告置信度)。但二者产生这一报告的机制并不对称:人的置信度来自真实的二阶内省,模型的置信度则由其生成过程产出。因此本研究能够主张的是行为层面的元认知校准差异,而非内在元认知机制的等同或缺失;这一边界将在讨论部分进一步展开。

**3.5 辅助分析:言行是否一致**

作为对主分析的补充,对于能够返回 token
概率(logprobs)的模型,本研究额外采集一份基于内部概率的置信度,用以考察同一模型\"verbalized
的主观置信度\"与\"内部 token

verbalized
数据的可信度;若不一致------即模型内部\"有所迟疑\"却在语言上表达自信------则本身构成一个值得关注的发现,呼应
Steyvers 等(2025)关于\"模型知道、却未如实传达\"的观察。由于部分 API
